1. make repo
    - gitignore
    - liscense
    - etc.
2. make vagrant file
    ‘vagrant init ubuntu/bionic64’
    - fill in starter text
    - run 'vagrant up' to start vagrant 
    - run 'vagrant ssh' to connect the vagrant server
        any commands on this server go to the guest OS
    - 'exit' to exit out of server.
3. use the server
    'cd /vagrant'
        - now projectt and server is linked
    - run code from project to server
        'python file_name.py'
4. setup the environment 
    - inside the /vagrant of server: 'python -m venv ~/env'
    - to activate venv: 'source ~/env/bin/activate'
    - to deactivate: 'deactivate'
5. install via requirements.txt 
    - inside /vagrant of server, make a equirements.txt file
    - put in dependencies
    - 'pip install -r requirements.txt'
6. create a new django app/project
    - inside /vagrant of server
        'django-admin.py startproject project_name .'
    - create app within project (project can have multiple apps, seperation 
        of functionality; microservice)
        'python manage.py startapp app_name'
7. enable app in django project
    - go to project/settings.py
    - put app inside:
    INSTALLED_APPS = [
    'django.contrib.admin',
    'django.contrib.auth',
    'django.contrib.contenttypes',
    'django.contrib.sessions',
    'django.contrib.messages',
    'django.contrib.staticfiles',
    ]
    - add: 'rest_framework', 'rest_framework.authtoken', and finally 'app_name',
8. run the app
    - 'python manage.py runserver 0.0.0.0:8000'
    - run server in browser: '127.0.0.1:8000'
    - to stop: ctrl+c
